# Quickstart: Voice-to-Text Input

This document provides a brief guide on how to use the new voice-to-text feature.

## How to Use

1.  **Navigate to the Main Screen**: Open the Personal Doctor AI application.
2.  **Locate the Microphone Button**: You will see a new microphone icon next to the text input field.
3.  **Start Recording**: Click the microphone button to start recording your voice. The icon will change to indicate that recording is active.
4.  **Speak Your Query**: Speak clearly into your microphone in either English or Bahasa Indonesia.
5.  **Stop Recording**: Click the microphone button again to stop recording.
6.  **Review Transcribed Text**: The system will transcribe your speech, and the resulting text will automatically appear in the input field.
7.  **Submit or Edit**: You can now submit the query to the AI assistant or manually edit the text if needed.
