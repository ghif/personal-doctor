# Data Model: TTS Summary Response

## Domain Entities

### AudioSummaryRequest
Represents the request to generate a spoken summary from a full text response.
- **original_text** (string): The full length text generated by the main MedGemma agent.
- **language** (string, optional): The target language (default: "en").

### SummaryAgentContext
Represents the context required by the ADK agent.
- **input_text** (string): The text to summarize.
- **constraints** (list): ["single paragraph", "no punctuation"]

## API Data Transfer Objects (DTOs)

### TTSRequest (Existing - Reused)
```json
{
  "text": "The patient shows signs of..."
}
```

### SummaryTTSResponse
The response is a binary stream (audio/wav), not a JSON object.
